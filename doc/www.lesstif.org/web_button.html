LessTif Web Button

       The LessTif team has created a web button that you can use to show your application no longer has to use Motif ® because of the availability of LessTif. Of course, even if you don't have an application, you are more than welcome to place the button and a link to the LessTif web site on your own home page.

To include the LessTif button on your web site, save the button shown here and reference it in your HTML like this:

<a href="http://www.lesstif.org"><img src="name_of_image.jpg"></a>
