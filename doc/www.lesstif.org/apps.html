Applications, libraries, ... that work with LessTif

This page is our success list:
it shows which software (applications, libraries, etc.) runs with LessTif! (and OSF/Motif®, of course :-)

As you can see it currently lists an impressive number of working applications. And we even don't list the numerous applications written for very special purposes which are not of use for anyone else. Some of these are pretty darned complicated, in user interface as well as in the application itself. ddd, xmcd, xephem, and NEdit are good examples of those. Unfortunately we are less than perfect, so we do have shorter lists of applications that don't work. Some of the applications/libraries listed as not working may actually work, this means we didn't get around to testing them lately and nobody sent us a note to say it does work.

If you know of software not currently listed below, then please send us a note about this, mentioning what category the application (or library) belongs in. If at all possible, please include an URL and a short description.

Maintaining this list of Motif applications is a time-consuming task. So please don't copy major parts of it without giving credits to us!
Thanks!
