LessTif News

LessTif Status
The current version of LessTif is 0.92.32 as of June 2, 2001. The code is available for download in various packages.


Latest LessTif News
We're updating our list of platforms on which you can use LessTif. Please take a minute and check whether you can add a new entry! We are looking for new combinations of architecture and operating system only, not just a slightly different setup than already mentioned.
There's an ugly but functional page on ftp.hungry.com download stats. On sourceforge.net, this is automatically provided by the SF software (look at the files section on the LessTif project pages). Also the old "nightly" script on ftp.hungry.com is now run hourly, and avoids sending lots of E-mail.
Xbae 4.9.1 - is released as of May 30, 2001. This version should be more stable than 4.9.0, and contains new features from the 4.8.* development releases (resizable rows, 24 bit color support, some bold column support) which should be considered stable.
We've started development on Motif and Dt printing API's. Read more about what they are, how to use them in your applications, which application demonstrates them, etc here.
The XbaeMatrix home page that used to be maintained by Andrew Lister has been recovered and is now available here and here.
The LessTif home page can now be found on two places : on http://www.lesstif.org and on http://lesstif.sourceforge.net. Both are automatic copies of files which are maintained in the CVS which also holds the sources.
The OpenGroup has published their sources of OpenMotif. Read the core team's statement why the LessTif project will have to stay alive!
